Barebones Explanation

From input source model and user choices, we derive 1+ of each kind:
  • explanation (txt)
  • diagram (svg)
  • in-math (expressions, equations, functions)
  • in-code (types, rules, functions)
  • in-param (numbers, choices, flags)
  • in-bulk-data (nominals, numbers, text)
Then we derive outputs below, based on inputs above:
  • out-math
  • out-bulk-data
  • out-plot (JSON -> Vega, d3, Plot.ly, other)
  • out-log
